# Power-Eater
  
This is Ni-MH battery charger for mini4wd.  
 - very low noise and low ripple  
 - dv charge / discharge  
 - cccvdv charge (i thinked new method for mini4wd charger)  
   - race charge method  
   - low internal resigtance, high voltage, many capacity  
   - low life  
 - programable charge/discharge sequence  
   - can use battery tester  
   - complex sequence like IF  
  
  
ミニ四駆用ニッケル水素充電器です。  
 - ノイズやリプルが少ない  
 - dv充電や放電ができる  
 - cccvdv充電ができる  
   - レース用途に適している  
   - dv方式に比べて、低内部抵抗、高電圧、大容量の充電が可能  
   - 寿命は短くなる  
 - 充放電シーケンスを組むことができる  
   - 簡易充放電試験機として使用可能  
   - プログラムのIF文のような分岐も可能
